Details of Online Grocery Store Shopping Project Website:-
This Online Grocery Store Shopping Project Website is based on Online Grocery Store where users can buy products for their daily needs. This is a fully dynamic project which is connected with a database. There are 2 modules in a project which are as follows:-

Admin Module of Grocery Store Website:- 
In this admin can add, update, delete categories, subcategories, brands, and products according to requirements. As this is a fully dynamic project whatever admin will add here it will show to user automatically. Admin can also view orders received, messages and feedback from user. All the admin pages are secured and cannot be opened without admin login. 

User Module:- User can signup and login into the website to buy the products. All the content shown on user side is dynamic like categories, sub categories, products and product details shown are fetched from database. Even the categories, subcategories show in left side menu are also dynamic and connected to database.  

User can see product details and add their desired products to cart. There are various payment options which one use for making payment. After making payment user will get order number and its details. User can also see their order details from order history. User can also search for their desired products by various search options available in site. Users can also change their password whenever needed.

After order, user receives an invoice at registered email address and a message with order details on the registered mobile number.

The GroMobile app and EcommerceWebsite are linked through a shared backend system that manages products, users, orders, and payments. Both platforms access the same database and APIs, ensuring that any updates made on one side (like adding products, processing orders, or updating user profiles) are instantly reflected on the other. Users can log in with a single account, browse products, add items to their cart, and complete purchases seamlessly whether they are on the mobile app or the website. This creates a unified shopping experience, with synchronized data, consistent design elements, and shared business logic.
